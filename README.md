# EcoÁgua

## Sistema de Orientação para Economia de Água Residencial

## Sobre o Projeto

Atividade da matéria de Algoritmos e Programação.

O **EcoÁgua** é um trabalho desenvolvido em linguagem C para uma atividade da Pós-Graduação em Desenvolvimento de Sistemas Web e Aplicativos Móveis.

O objetivo do projeto é criar uma solução simples para medir e analisar o consumo de água em residências, ajudando os moradores a identificarem se o consumo está adequado, em nível de atenção ou alto.

A proposta busca aplicar conceitos básicos de programação em C, utilizando estrutura sequencial e comandos condicionais, para resolver um problema real relacionado ao uso consciente da água.
